저의 고향은 전라남도 순천입니다.

제 별명은 토마토입니다.

저는 국어국문학을 전공하며 다양한 경험을 통해 창의력과 논리적 사고를 다져왔습니다. 예체능 동아리에서 치어리딩과 연극 활동을 하며 협업 능력과 소통 능력이 강점이 되었고, 연기와 방송에 대한 관심도 더욱 확고해졌습니다. 중국어와 중국 문화에 대한 흥미로 튜터링 활동을 시작하며 언어 능력과 문화적 이해를 확장했고, 서빙 아르바이트를 하며 체력적으로 힘든 상황 속에서도 인내심과 시간 관리 능력을 발휘했습니다. 대학글쓰기 과목에서 A+를 받으며 글쓰기 능력과 논리적 사고에서 성과를 내었고, 모든 경험이 저를 더 성장시키는 원동력이 되었습니다.
