# tomato
조이은의 자기 소개 페이지 입니다.

저의 고향은 전라남도 순천입니다.

제 별명은 토마토입니다.

저는 국어국문학을 전공하며, 예체능 동아리 활동을 통해 협업과 리더십을 적극적으로 배워왔습니다. 치어리딩과 연극 활동에서 다양한 사람들과의 소통을 통해 팀워크를 쌓고, 리더로서의 역할도 경험하면서 협업 능력이 강점으로 자리 잡았습니다. 중국어에 대한 흥미로 튜터링 활동을 시작하며 언어 능력을 확장했고, 이를 통해 글로벌 시각과 문화적 이해가 더욱 넓어졌습니다. 서빙 아르바이트를 하며 체력적인 어려움 속에서도 성적 향상을 위해 지속적으로 노력한 경험을 통해 인내심과 시간 관리 능력을 키웠습니다. 또한, 대학글쓰기 과목에서 높은 성적을 달성하기 위해 주제에 대한 깊이 있는 분석과 체계적인 글쓰기를 반복하며 비판적 사고와 문제 해결 능력을 키운 과정에서, 철저한 준비와 성실함이 제 큰 강점임을 실감했습니다.

"글과 관계를 통해 세상과 소통하고 성장하는 사람"
